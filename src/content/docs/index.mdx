## Passo-a-Passo para Abertura de Chamados:

import { Steps } from '@astrojs/starlight/components';

<Steps>

1. Acesse o [Nexus](https://nexus.lojasmm.com.br).

2. Realize seu login com as mesmas credenciais que utiliza em seu email.

3. Escolha a aba `Formulários` e selecione o setor que irá realizar o atendimento.

4. Selecione a categoria que mais se encaixa com seu problema e descreva-o.

</Steps>
