O ATA GKM 2210T da Intelbras é um Adaptador de Telefone Analógico (ATA) projetado para converter sinais de telefones analógicos em dados digitais, permitindo que esses aparelhos se conectem a redes de telefonia VoIP (Voice over IP). Ele é ideal para usuários que desejam utilizar serviços VoIP, mas preferem continuar usando seus telefones convencionais.

### Conexões Importantes
1. Entrada para o cabo do telefone (TEL 2): Caso haja um segundo telefone, conecte-o aqui.
2. Entrada para o cabo do telefone (TEL 1): Utilize essa porta para conectar o primeiro telefone analógico.
3. Entrada para o cabo de rede mas a porta LAN **NÃO É UTILIZADO**.
4. Entrada para o cabo de rede/internet: Conecte o cabo de rede (geralmente de cor azul) que levará o sinal de internet para o ATA.
5. Entrada para o cabo de energia: Conecte o cabo de energia do aparelho nessa entrada para ligá-lo.

### Passos para Instalação

1. Conectar o cabo de rede: Conecte o cabo de rede à porta **WAN** e ao roteador, switch ou tomada da rede local.
2. Conectar o telefone: Utilize as portas de telefone para conectar os aparelhos analógicos.
3. Ligar o ATA: Insira o cabo de energia e verifique se as luzes indicadoras estão acesas corretamente.

### Resolução dos Problemas Comuns

Se o telefone não estiver funcionando, siga as instruções:

#### 1. Reiniciar o ATA

- Solução mais comum: Desligue o aparelho da tomada, aguarde 5 minutos e conecte-o novamente. Isso resolve a maioria dos problemas temporários.

#### 2. Abrir um chamadp no HelpDesk

- Se o problema persistir, pode ser um defeito técnico. Nesse caso, abra um chamado no Nexus para que a equipe possa verificar o equipamento.
