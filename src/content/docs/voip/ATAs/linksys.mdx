O ATA Linksys (Adaptador de Telefone Analógico) é um dispositivo que conecta telefones analógicos tradicionais a uma rede de telefonia digital VoIP (Voice over IP). Ele atua como um conversor entre o sinal analógico e o sinal digital da internet, oferecendo uma solução econômica e eficiente para a comunicação telefônica em filiais e escritórios.

### Conexões Importantes
1. Entrada para o cabo do telefone (Porta 1): Utilize essa porta para conectar o primeiro telefone analógico.
2. Entrada para o cabo do telefone (Porta 2): Caso haja um segundo telefone, conecte-o aqui.
3. Entrada para o cabo de rede/internet: Conecte o cabo de rede (geralmente de cor azul) que levará o sinal de internet para o ATA.
4. Entrada para o cabo de energia: Conecte o cabo de energia do aparelho nessa entrada para ligá-lo.

### Passos para Instalação

1. Conectar o cabo de rede: Conecte o cabo de rede à porta de internet e ao roteador, switch ou tomada da rede local.
2. Conectar o telefone: Utilize as portas de telefone para conectar os aparelhos analógicos.
3. Ligar o ATA: Insira o cabo de energia e verifique se as luzes indicadoras estão acesas corretamente.

### Resolução dos Problemas Comuns

Se o telefone não estiver funcionando, siga as instruções:

#### 1. Reiniciar o ATA

- Solução mais comum: Desligue o aparelho da tomada, aguarde 5 minutos e conecte-o novamente. Isso resolve a maioria dos problemas temporários.

#### 2. Abrir um chamadp no HelpDesk

- Se o problema persistir, pode ser um defeito técnico. Nesse caso, abra um chamado no Nexus para que a equipe possa verificar o equipamento.

### Conclusão

Este guia fornece as etapas básicas de instalação e solução de problemas para os ATAs Linksys nas filiais. Se houver qualquer outro problema que não possa ser resolvido com essas medidas, contate a equipe do  HelpDesk.
