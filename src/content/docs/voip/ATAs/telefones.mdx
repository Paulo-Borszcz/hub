## Introdução

A configuração de telefones fixos nas filiais envolve alguns passos simples para a conexão de cabos, visualização do endereço IP e instruções de ligação. Este tutorial irá guiá-lo através desses processos.

## Conexão de Cabos

**Passo a passo:**

Conecte os seguintes cabos no telefone:
   - **LAN** (Cabo de Internet)
   - **DC5V** (Energia)
   - **Telefone** (Handset)

## Visualizar o IP

**Passo a Passo:**

<Steps>
1. Pressione o primeiro de três botões no painel do telefone.
2. O telefone irá exibir o endereço IP.
</Steps>

<Aside type="caution">
Caso o IP exibido seja `0.0.0.0`, isso provavelmente indica um erro na conexão de internet cabeada. Nesse caso, recomendamos utilizar um cabo de rede de uma máquina do caixa para verificar se o telefone volta a funcionar. Se retornar, significa que o problema estava no cabo de rede original. Nesse caso, será necessário solicitar um novo cabo de rede.
</Aside>

## Realizar Ligações


1. Para ligações onde o DDD é igual entre ambas as partes, basta digitar `0` seguido do número de telefone. Exemplo: `0-98919292`.

2. Para ligações para um DDD diferente, digite `0(DDD)` seguido do número de telefone. Exemplo: `0-43-98919292`.
